# 🚀 Things I've Built

### 🚗 Car Rental System
> Microservices · Spring Boot · RabbitMQ · Redis · Docker · MongoDB · Netflix Eureka · Spring Cloud Gateway

A full microservices platform for car rentals — not just a CRUD app.
Built with distributed locking in Redis to prevent double bookings, async payment processing via RabbitMQ with Dead Letter Queues, and JWT auth with refresh token rotation validated at the API Gateway. Covered with 35+ unit tests and containerised end-to-end with Docker Compose.

### 🏙️ City Relocator App
> Spring Boot · WebFlux · MongoDB Atlas · Redis · Google Places API · React Native · Docker

A mobile-first city relocation assistant. Uses Spring WebFlux with `Flux.merge` to fetch multiple place categories concurrently, Redis geo-spatial caching with `GEORADIUS` to cut redundant API calls, and persists results to MongoDB Atlas. Backend deployed on Render, Android APK distributed via Expo EAS Build.
